Ask yourself: Is the sentence a fragment?

To decide if a sentence is a fragment, ask yourself:

Can I find a subject?

If you answer no, the sentence is a fragment and needs to be fixed.

If you answer yes, next ask yourself:

Can I find a verb?

If you answer no, the sentence is a fragment and needs to be fixed.

If you answer yes, next ask yourself:

Is the sentence a dependent clause without an independent clause to complete the idea?

If you answer yes, the sentence is a fragment and needs to be fixed.

If you answer no, the sentence is not a fragment!
