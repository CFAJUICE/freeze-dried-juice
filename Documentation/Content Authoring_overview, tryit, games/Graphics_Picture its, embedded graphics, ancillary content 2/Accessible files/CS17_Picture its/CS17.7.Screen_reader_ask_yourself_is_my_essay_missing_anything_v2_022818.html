Ask yourself: Is my essay missing anything?

Don't forget to compare your work to the assignment or rubric criteria to make sure you didn't forget anything. Begin by asking yourself:

Did I follow directions?

If you answer no, you will need to make more revisions. An essay that doesn't follow all of the rules of the assignment will not be successful.

If you did follow all of the directions, ask yourself:

Did I use available resources?

If you answer no, you will need to make more revisions. You may have missed something important in a resource.

If you did use all available resources, ask yourself:

Does my work include everything identified in the rubric?

If you answer no, you will need to make more revisions. You don't want to ignore important requirements of an assignment.

If you did include everything identified in the rubic, ask yourself:

Did I apply any feedback that I was given?

If you answer no, you will need to make more revisions. Revising your essay to reflect feedback will make it better.

If you applied all of the feedback you were given, Great! You're ready for the final step—proofreading and editing your essay!
