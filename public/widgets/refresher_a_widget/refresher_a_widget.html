This is your first day on the job! Your first task is to read and edit Letters to the Editor. You select the first letter. It seems like a great letter to print, but you think it might have a run-on sentence.

How can you tell and how can you fix it?

Dear Editor,

Your article about the dangers of sharing on Facebook was great! Everyone in my workplace read it and agreed. Social media can be dangerous there are so many ways it can hurt you if you are not cautious. We want to see more articles like this!

First, decide if the sentence is a run-on sentence.
A run-on sentence has two separate sentences joined as one. It sounds right in speech, but it is not correct.

To decide if itâ€™s a run-on sentence, figure out if it has two independent clauses that can stand alone.
Social media can be dangerous there are so many ways it can hurt you if you are not cautious.
Then, fix the sentence in one of three ways.
1. Add end punctuation to create two sentences
Social media can be dangerous. There are so many ways it can hurt you if you are not cautious.
2. Add a comma and a joining word like and between the two clauses
Social media can be dangerous, and there are so many ways it can hurt you if you are not cautious.
3. Add a semicolon between the two clauses
Social media can be dangerous; there are so many ways it can hurt you if you are not cautious.
Which correction should you choose for this letter? All three are good choices! Try using them all to add variety to your writing.
