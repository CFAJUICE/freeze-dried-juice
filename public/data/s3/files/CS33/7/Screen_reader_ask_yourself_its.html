Ask yourself: Deciding if you should use an apostrophe with "its"

To decide if you should use I T S, "its," or, I T apostrophe S, "it's," ask yourself:

Do I want to show possession?

Example: The cat drank the cat's milk.

If you answer yes, use I T S, "its". Example: the cat drank, I T S, its milk.

If you answer no, next ask yourself:

Do I mean it is?

Example: It is raining outside.

If you answer yes, use I T apostrophe S, "it's". Example: I T apostrophe S, It's raining outside.
